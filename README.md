A comprehensive, AI-powered agricultural advisory platform designed to empower farmers with data-driven insights, real-time pest identification, and localized market information.

---

## 🚀 Key Features

### 🤖 AI-Powered Advisory
- **Precision Crop Recommendations:** Get personalized crop suggestions based on your soil type, PH levels, and location.
- **AI Insights Dashboard:** Real-time analysis of farm data using advanced AI models.

### 🛡️ Pest Management & Identification
- **Visual Identification:** Upload images of crop leaves for instant AI-based pest and disease diagnosis.
- **Treatment Guides:** Access step-by-step instructions on dosage and frequency for recommended pesticides.
- **Community Reporting:** Report pest sightings to help nearby farmers stay prepared.

### 🍱 Multi-Language Support
Comprehensive support for **10 languages** to ensure accessibility for every farmer:
- 🇺🇸 English &nbsp; 🇮🇳 Hindi &nbsp; 🇮🇳 Punjabi &nbsp; 🇧🇩 Bengali
- 🇮🇳 Telugu &nbsp; 🇮🇳 Tamil &nbsp; 🇮🇳 Gujarati &nbsp; 🇮🇳 Marathi
- 🇫🇷 French &nbsp; 🇪🇸 Spanish

### 💬 Intelligent Farm Assistant
- Voice and text-enabled chatbot for instant farming queries.
- Powered by Gemini AI for deep agricultural knowledge.

## 📦 Installation & Setup

1. **Clone the repository:**
   ```bash
   git clone https://github.com/ankitha-pidugu/Smart-Crop-Advisory-System.git
   cd Smart-Crop-Advisory-System
   ```

2. **Install dependencies:**
   ```bash
   npm install
   ```

3. **Environment Setup:**
   Create a `.env` file in the root directory and add your credentials:
   ```env
   VITE_SUPABASE_URL=your_supabase_url
   VITE_SUPABASE_ANON_KEY=your_supabase_anon_key
   VITE_GEMINI_API_KEY=your_gemini_api_key
   VITE_WEATHER_API_KEY=your_weather_api_key
   ```

4. **Start Development Server:**
   ```bash
   npm run dev
   ```
